# Project
    -- A Full-Stack E-commerce Website 

## To run:

1. Create Virtual Environment:
    > pip install pipenv
    > pipenv shell

2. Install Packages:
    > pip install -r requirements.txt

3. Create Superuser
    > py manage.py createsuperuser

4. Connect Database:
    > Open phpmyadmin and create a DB called "shopkartdb"
